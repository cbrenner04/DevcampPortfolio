# DevcampPortfolio

https://www.udemy.com/professional-rails-5-development-course

> This is a Ruby on Rails 5 application that allows users to create their own
  portfolios.

## Features

- Real time chat engine for comments
- Blog
- Portfolio
- Drag and drop interface
